# Go言語（Golang）のインストール・セットアップ・実行ガイド

このガイドでは、Go言語のインストールから、サンプルの `hello-world` プログラムを実行するまでの手順を初心者向けに解説します。

## 1. Go言語のインストール

まず、お使いのパソコンにGo言語をインストールする必要があります。

### macOSの場合

以下の2つの方法のどちらかでインストールするのがおすすめです。

**方法A: 公式サイトからインストーラーを使う（推奨）**
1. [Go公式サイトのダウンロードページ](https://go.dev/dl/) にアクセスします。
2. `Apple macOS` 用のインストーラー（例: `go1.xx.x.darwin-amd64.pkg` または `arm64.pkg`）をダウンロードします。
   - ※ M1/M2/M3チップ搭載のMacなら `ARM64`、Intelチップなら `x86-64` を選びます。
3. ダウンロードしたファイルを開き、画面の指示に従ってインストールします。

**方法B: Homebrewを使う（コマンドラインに慣れている方向け）**
ターミナルで以下のコマンドを実行します。
```bash
brew install go
```

### インストールの確認

インストールが完了したら、ターミナル（CursorのターミナルでもOK）を開いて、正しくインストールされたか確認します。

```bash
go version
```

これ実行して、`go version go1.xx.x darwin/arm64` のようにバージョン情報が表示されれば成功です！

## 2. 開発環境のセットアップ

### エディタの拡張機能

Cursor（またはVS Code）を使用している場合、Go言語の開発を快適にするために「Go」拡張機能をインストールすることをおすすめします。

1. 拡張機能メニュー（左側のブロック崩しのようなアイコン）を開きます。
2. 「Go」と検索します。
3. `Go Team at Google` が提供している拡張機能をインストールします。

これにより、コードの補完やエラーチェックが自動で行われるようになります。

## 3. サンプルプログラムの実行

実際に、このプロジェクトに含まれている `example/ch01/hello-world` のコードを実行してみましょう。

### 手順1: 対象のディレクトリに移動する

まず、ターミナルでサンプルコードがあるフォルダに移動します。
現在 `lgo2` フォルダ（プロジェクトのルート）にいると仮定して、以下のコマンドを入力します。

```bash
cd example/ch01/hello-world
```

### 手順2: プログラムを実行する (`go run`)

Goのプログラムを実行する一番簡単な方法は `go run` コマンドです。これはコンパイルと実行を一度に行います。

```bash
go run hello.go
```

**実行結果:**
```
Hello, world!
```
と表示されれば成功です！

### 手順3: 実行ファイルを作成する (`go build`)

プログラムを配布したり、単体で動く「実行ファイル」を作りたい場合は `go build` を使います。

```bash
go build
```

これを実行すると、同じフォルダに `hello-world`（Windowsなら `hello-world.exe`）という名前のファイルが作られます。
これを実行するには以下のようにします。

```bash
./hello-world
```

これでも同じように `Hello, world!` と表示されます。

## 4. コードの解説

実行した `hello.go` の中身を少しだけ解説します。

```go
package main // 1. パッケージ宣言

import "fmt" // 2. インポート宣言

func main() { // 3. メイン関数
	fmt.Println("Hello, world!")
}
```

1.  **`package main`**: このファイルが「メイン」のプログラムであることを宣言しています。実行可能なプログラムを作るには必ず `package main` が必要です。
2.  **`import "fmt"`**: 文字を表示したりするための便利な機能（パッケージ）を取り込んでいます。`fmt` は "format" の略で、入出力を扱います。
3.  **`func main() { ... }`**: プログラムはここから動き始めます。この `main` 関数の中に書かれた命令が上から順に実行されます。
    - `fmt.Println("...")`: 画面に文字を表示して改行する命令です。
