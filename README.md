# 初めてのGo言語 第2版

<img src="https://www.marlin-arms.com/jpn/arts/books-small/learning-go2.png?v=2" width="200px">

[オライリー・ジャパン](https://www.oreilly.co.jp/)発行の『[初めてのGo言語 第2版](https://www.marlin-arms.com/support/learning-go2/)』（2025年8月初旬発売予定）の例題および練習問題の解答例のリポジトリです。

## 学習者の方へ：このリポジトリの使い方

このリポジトリは書籍『初めてのGo言語 第2版』の学習をサポートするために作られています。
Go言語の基礎から応用まで、以下のステップで学習を進めることができます。

### 1. 環境構築と実行方法
まずはGo言語の開発環境を整えましょう。詳細な手順は以下のガイドを参照してください。
- [**Go言語インストール・セットアップ・実行ガイド**](./Setup.md)

### 2. 章ごとの学習ガイド
各章の `example` フォルダには、書籍で紹介されているサンプルコードが含まれています。
初学者向けに、各章のポイントとコードの解説をまとめたガイドを用意しています。

- [**第1章: Go環境のセットアップと最初のプログラム**](./example/ch01/README.md)
- [**第2章: 基本型と宣言**](./example/ch02/README.md)
- 第3章: 合成型（準備中）
- ...

### 3. リポジトリの構造
- **example/**: 書籍内のサンプルコード（写経用、動作確認用）
- **exercise/**: 練習問題の解答例
